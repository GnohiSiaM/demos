# Java

## 安装

```sh
# 安装 jenv jenv
$ curl -L -s get.jenv.io | bash
# 更新 jenv:          jenv selfupdate
# 更新 repository:    jenv repo update

# 查找 candidates 可用的版本:
$ jenv ls java
# 安装 candidates [JDK, Maven, SpringBoot...]
$ jenv install java 1.8.0_91
$ jenv install maven 3.3.9
$ jenv install boot 1.4.1
```
